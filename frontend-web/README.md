Esqueleto de frontend web (Vite + React)

Estrutura:
- frontend-web/
  - package.json
  - src/
    - main.jsx
    - App.jsx
    - api.js (chama backend)
    - pages/
      - Home.jsx
      - Recommend.jsx

Notas:
- Execute `npm install` e depois `npm run dev` em `frontend-web` quando Node/npm estiver disponível.
- Atualize `api.js` para apontar para sua URL de backend (padrão: http://localhost:4000/api).
