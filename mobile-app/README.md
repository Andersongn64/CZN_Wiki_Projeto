Esqueleto de app mobile (Expo React Native)

Estrutura:
- mobile-app/
  - app.json
  - package.json
  - App.js
  - src/
    - api.js (mesma interface de API do web)
    - screens/
      - Home.js
      - Recommend.js

Notas:
- Use Expo Snack ou Expo CLI local para rodar o app. O app mobile usa os mesmos endpoints do backend.
- Ao usar Expo Snack, cole os arquivos no editor e execute em web ou dispositivo.
